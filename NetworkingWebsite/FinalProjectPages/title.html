Networking and Parallel Final Project Spring 2017

Dynamic Lighting for Large-Scale Map Ray Tracing

  • Grant Moe

  • Buster Schrader

  • Jordan Falcon

We will be implementing a large scale solution to applying ray tracing for all models in a single model. Previously the best way to apply ray tracing in the source engine was to simply bake each and every model in a map with ray tracing lines which can be tedious and time consuming and custom scripts. We will be using the NVIDIA Titan GPU to implement our parallized solution to increase the speed for ray tracing the entire model.

Author: Grant Moe

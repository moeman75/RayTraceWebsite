Networking and Parallel Final Project Spring 2017

    PLAN TO ACHIEVE

  • input and output for entire map to then be ray traced
  • better and faster than putting every model through ray tracing function already made for game maker
  • puts less load on the CPU which would usually ray trace the lines
  • uses CUDA kernals to compute the polygons and intersections for ray tracing lines for any models in a map for a desired ray being produced
  • run our program on the NVIDIA titan

    HOPE TO ACHIEVE

  • modular design and easily used
  • uploading maps and displaying processes being run and what is happening to ray trace the map
  • under .5 seconds to ray trace a map of reasonable size with CUDA
  • stand alone program that can be run on multiple different machines if there is a NVIDIA graphics card installed onboard, and CUDA installed
  • map making architects dreams will be made
